# Интеграция языковых моделей ollama 

## Получение бинарного ответа на вопрос (binary_answer_function и binary_answer_logic)
Данный код создан для автоматизации обработки текстов. 
В файле binary_answer_function содержится функция, которую можно интегрировать в код на различных этапах;
В файле binary_answer_logic - модель не собрана функцию - 
и есть несколько поинтов, где можно отследить логику работы функции binary_answer_function;
посмотреть, как мыслит модель.
Также реализована трассировка.

## Обработка текстовой информации, краткая логическая выжимка
В коде реализована обработка текста с помощью ollama.
Модель на вход получает текст. Выделяет из него самую значимую информацию.
Возвращает: 
* summary - краткая сводка (1-2 предложения);
* список - bullets - несколько ключевых пунктов;
* keywords - ключевые слова.

Для примера работы функции реализованы три метода получения текстовой информации:
VK api, TG api и парсинг статьи с новостного веб-сайта.

## Выводы модели ollama о результатах OLS (regression_analysis)
В данном коде реализована простая OLS-регрессия, 
получение базовых метрик оценки качетсва регрессионного анализа
и обращение к локальной LLM семейства Llama (разработанных компанией Meta)
для простой оценки результатов OLS и качества модели регрессии.

Код гибкий и может быть использован с собственными данными:
тогда необходимо удалить блоки "Определение показателей и стран" и "Вызов функции, получение данных"
и создать два словаря - один с ключом и значением целевого показателя indicator_y={}, 
второй - с клбчами и значениями перменных indicators_x={}
и список с названиями экономических субъектов, по которым проводится анализ,
в данном примере это старны - country_list=[].

В коде же, в блоках "Определение показателей и стран" и "Вызов функции, получение данных" для формирования тестовой выборки 
используется обращение World Bank API - 
реализация аналогично, гибкая, и поддерживает изменение (в рамках логики World Bank API)
исследуемых параметров, стран из группы исследования, а также количества переменных.
